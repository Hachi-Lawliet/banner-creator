# BannerCreator デザインシステム

## 1. デザイントーン

- クリーン・ミニマル・プロフェッショナル
- ツール系UI（Figma, Canvaを参考）
- 入力パネルは控えめ、プレビューが主役

## 2. カラーパレット

### ツールUI色（バナー生成色とは別）

| 名前 | HEX | 用途 |
|------|------|------|
| Background | `#F8F9FA` | アプリ背景 |
| Surface | `#FFFFFF` | カード・パネル背景 |
| Foreground | `#1A1A2E` | テキスト（メイン） |
| Muted | `#64748B` | テキスト（補助） |
| Primary | `#2563EB` | アクセント・ボタン |
| Primary Light | `#DBEAFE` | ホバー・選択状態 |
| Border | `#E2E8F0` | ボーダー・区切り線 |
| Error | `#EF4444` | エラー表示 |
| Success | `#22C55E` | 成功表示 |

### 使い分けルール
- メインカラー: Foreground (70%)
- サブカラー: Background + Surface (20%)
- アクセント: Primary (10%)

| トークン | 値 | 用途 |

## 6. レイアウトルール

### ヘッダー
- 高さ: 56px
- 背景: Surface
- 下ボーダー: 1px solid Border
- アプリ名: 左寄せ、700ウェイト、18px
- ダウンロードボタン: 右寄せ

### 入力パネル
- 幅: 360px 固定
- 背景: Surface
- 右ボーダー: 1px solid Border
- スクロール: 縦スクロール
- パディング: 16px

### プレビューエリア
- 背景: Background
- バナーは中央配置
- プレビュー下に情報表示（サイズ・重心・コントラスト比）

## 7. アイコン

- アイコンライブラリは使用しない（軽量化のため）
- 必要な場合はSVGインラインで最小限のアイコンのみ
- ダウンロードアイコン、アップロードアイコン程度
